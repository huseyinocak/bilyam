# Release Checklist (MVP)

- [ ] Feature + unit tests pass on CI.
- [ ] No auth/ownership regression on customer/admin routes.
- [ ] Guest and authenticated quote flows validated.
- [ ] Queue/mail config changes reviewed (`config/queue.php`, `config/mail.php`).
- [ ] Migration impact reviewed (forward + rollback risk).
- [ ] Staging smoke test completed for critical releases.
- [ ] Post-release smoke checks planned (catalog, quote submit, admin dashboard).
