# Branding Assets

## Dosya Konumu
Tüm branding dosyaları `docs/assets/branding/` altındadır.

## Kullanım Kuralları
- `toptanbilyalogo.png`: ana logo
- `favicon.png`: modern favicon kaynağı

## Codex Talimatı
1. Branding dosyalarını projede uygun yerlere entegre et.
2. Web uygulamasında favicon ve gerekirse apple-touch-icon tanımla.
3. Uygulama içinde header/sidebar/login sayfalarında logo kullanımını standardize et.
4. Dosya isimlerini değiştirme; mevcut isimleri referans al.
5. Yeni türev asset gerekiyorsa mevcut dosyalardan üret ama orijinal dosyaları bozma.
